# Across



## <<给SSR加速！只要一分钟即可>>
ref: https://www.liaosam.com/speed-up-your-ssr.html


**使用方法**

1. 下载 BBR 加速脚本：
```
wget --no-check-certificate https://github.com/teddysun/across/raw/master/bbr.sh
```

2. 执行：
```
chmod +x bbr.sh && bash bbr.sh
```

3. 用命令检查一下：
```
ps aux | grep bbr*
```


发现 bbr 的进程正在运行。

BBR 安装配置完毕，SSR 立即得到提速。
